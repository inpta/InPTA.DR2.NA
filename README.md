# InPTA.DR2.NA
## INPTA DR2 Pulsars – DM12 Noise Model Parameter Files

This repository contains the DM12 noise model parameter (.par) files derived from the noise analysis of pulsars in the Indian Pulsar Timing Array (InPTA) Data Release 2 (DR2).

📄 Associated Publication

The results presented here are based on the analysis published in:
Noise characterization of InPTA DR2 pulsars
Available at: https://www.sciencedirect.com/science/article/abs/pii/S2214404826000467

Please cite this paper if you use the parameter files from this repository in your work.

📦 Contents
*.par files:
Parameter files containing dispersion measure (DM) noise model components (DM12 model) for individual pulsars. Each file corresponds to a single pulsar and includes the best-fit parameters inferred from Bayesian noise modeling.


⚙️ Usage
These .par files can be used with standard pulsar timing software such as:
- Tempo2
- enterprise
- libstempo

📌 Notes
These files represent post-analysis products and are not raw observational data.
Users should ensure compatibility with their specific analysis pipeline.
No guarantees are made regarding direct plug-and-play usage without adjustment.
